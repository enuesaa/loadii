# loadii
Instant web server for development purposes.

## Usage
```console
$ loadii --help
Instant web server for development purposes

USAGE:
  loadii [global options] [arguments...]

FLAGS:
  --port value           Port (default: 3000)
  --dir value, -d value  Serve dir (default: ".")
  --help, -h             show help
  --version, -v          print the version
```
